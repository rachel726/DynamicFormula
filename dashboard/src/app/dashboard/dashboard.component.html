🔬 השיטות

1 SQL — Stored Procedure

חישוב SET-BASED מלא בתוך ה-DB באמצעות sp_executesql. INSERT...SELECT עם CASE WHEN דינמי — מעבד מיליון רשומות בפעולה בודדת. אין מעברים בין תהליכים, אין תעבורת רשת — הכל בתוך SQL Server.

2 C# — Compiled Expression Trees

מפרש נוסחאות עצמאי שבנינו, הממיר כל ביטוי ל-Func<double,double,double,double,double> דרך System.Linq.Expressions. קומפילציה פעם אחת ל-IL, ואז מיליון הפעלות במהירות native. משולב עם Parallel.For לניצול כל ליבות ה-CPU.

3 Node.js — math.js (Server-side JavaScript)

סקריפט node calculate.js הרץ על Node.js. משתמש ב-math.compile() לקומפילציה מראש של כל ביטוי, ואז מריץ מיליון חישובים בלולאה עם performance.now() למדידה מדויקת.
